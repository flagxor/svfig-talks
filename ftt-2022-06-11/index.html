CRYPTOGRAPHIC HASHING
---------------------
SHA1 (Secure Hash Algorithm 1)
  * 64 byte blocks
  * 20 byte digest
  * Developed by NSA
  * Used by Git
SHA256 (really SHA-2 Secure Hash Algorithm 2)
  * 64 byte blocks
  * 64 byte digest
  * Developed by NSA
  * Used by Bitcoin
